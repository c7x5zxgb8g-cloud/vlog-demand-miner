## 一句话介绍

NextTake 让同赛道的公开证据和上一条内容的真实结果，共同决定创作者下一条拍什么、怎么拍。

## 简短介绍

NextTake 是面向个人短视频创作者的证据驱动 Agent Skill。它从同赛道公开视频与评论中发现反复出现的受众问题，把证据转成选题、Hook、60–90 秒脚本、镜头与发布文案；创作者手动发布后，系统将真实播放、互动和评论与发布前预测对照，更新受众画像，推荐下一条并生成下一期完整文案。

## 核心能力

- **发现值得拍的内容**：从公开视频、字幕和评论中提取 Evidence，保留支持证据、反证与样本限制。
- **生成真正可拍的方案**：产出标题、Hook、脚本、镜头、发布文案与 CTA，并完成稿件评分和发布前预测。
- **把发布结果变成学习**：复盘播放、互动与评论，判断发布前假设是被验证、被推翻还是暂时无法判断。
- **把学习带进下一条**：更新受众画像，推荐下一选题并生成下一期完整文案，通过 Creator Studio 展示完整闭环。

## 当前验证与边界

- 2026-07-22 当前仓库 `96` 项自动测试通过；离线演示可生成 `4` 个内容机会与完整 Creator Studio。
- 演示中的 Discover 使用真实试点脱敏 Evidence；发布和表现数据会明确标记为演示数据。
- NextTake 不自动上传视频，拍摄与平台发布由创作者手动完成。
